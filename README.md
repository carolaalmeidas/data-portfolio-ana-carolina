# 🚕 NYC Taxi Data Analysis

## 📌 Objective
Analyze taxi trip data from New York City to identify patterns in trip behavior, pricing, and user activity.

## 📊 Dataset
- 408,294 rows (each row = one trip)
- 18 columns
- Source: NYC Taxi & Limousine Commission

## 🔍 Key Tasks
- Data cleaning and inspection
- Handling missing values
- Feature engineering
- Exploratory analysis

## ⚙️ Tools
- Python (Pandas, NumPy)

## 🚀 Key Insights
- Average trip distance and duration
- Fare patterns based on trip length
- Tip behavior analysis
- Peak usage patterns

## 📈 Features Created
- Trip duration (minutes)
- Fare per mile

## 💡 Business Value
Understanding trip patterns can help:
- Optimize pricing strategies
- Improve driver allocation
- Enhance customer experience
